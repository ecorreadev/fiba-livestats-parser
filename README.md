# FIBA LiveStats Parser

Pega un link de FIBA LiveStats y devuelve un JSON normalizado.

## Trazabilidad de versión

La versión visible en la página se define de forma estática en [public/index.html](public/index.html), en el texto `Versión en producción: vYYYY.MM.DD`.
Cada vez que se haga un cambio de código que vaya a producción, actualizar ese valor manualmente.

## Uso local

```bash
npm install
npm start
